# Fitness Tracker

A comprehensive fitness tracking application built with Kotlin to help users monitor their health and fitness goals.

## Features

- **Activity Tracking**: Log and track various physical activities and workouts
- **Progress Monitoring**: Visualize your fitness progress over time
- **Goal Setting**: Set and manage personal fitness targets
- **Data Insights**: Get analytics on your performance and trends
- **User-Friendly Interface**: Intuitive design for easy navigation

## Tech Stack

- **Language**: Kotlin
- **License**: GNU General Public License v3.0

## Getting Started

### Prerequisites

- Kotlin 1.x or higher
- Java Development Kit (JDK) 11 or later

### Installation

1. Clone the repository:
```bash
git clone https://github.com/al320/fitness-tracker.git
cd fitness-tracker
```

2. Build the project:
```bash
./gradlew build
```

3. Run the application:
```bash
./gradlew run
```
